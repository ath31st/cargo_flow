# Gateway Service

## Overview

The Gateway Service acts as an entry point to the system, routing requests to the appropriate
microservices based on their roles. It uses Spring Cloud Gateway and integrates with Keycloak for
authentication and authorization.

## Prerequisites

- Java 17 or higher
- Maven 3.6.3 or higher
- Docker

## Technologies Used

- Spring Boot: Application framework
- Spring Cloud Gateway: API gateway
- Spring Security OAuth2: Security and authorization
- Keycloak: Identity and access management
- Maven: Dependency management and build tool
